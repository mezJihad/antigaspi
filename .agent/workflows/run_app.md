---
description: Run the Antigaspi full stack application (Client & Backend)
---

1. Start the .NET Backend
   - This command starts the API server. It should be run in the background.
   
   cd backend-dotnet
   dotnet run --project Antigaspi.Api

2. Start the Vite Frontend
   - This command starts the frontend dev server.
   
   cd client
   npm run dev
